# Hospital Analysis Dashboard

## Project Overview
A dynamic Power BI solution designed for healthcare administrators to track patient care, hospital occupancy, and financial metrics. This dashboard integrates data from multiple departments to streamline hospital management and improve patient outcomes.

## Key Features
*   **Resource Management:** Real-time tracking of Doctor, Staff, and Patient counts.
*   **Bed Occupancy Tracking:** Visual donut charts for monitoring facility availability.
*   **Revenue Analysis:** Categorized breakdown of patient charges (Surgery, Room, Tests) to identify top revenue streams.
*   **Pharmacy Inventory:** A detailed heatmap for tracking medicine consumption by month and day.
*   **Appointment Scheduling:** Integrated view of upcoming patient-doctor consultations.

## Tech Stack & Skills
*   **Data Source:** MySQL Workbench (Relational Database Management)
*   **Visualization:** Microsoft Power BI
*   **Analytics:** DAX (Data Analysis Expressions) for calculating occupancy rates and discharge trends.
*   **Skillset:** Data Modeling, SQL Querying, Dashboard UI/UX Design.

## Data Pipeline
1.  **Extraction:** Data pulled from **MySQL Workbench** using optimized SQL queries.
2.  **Transformation:** Data cleaned and modeled in Power Query.
3.  **Visualization:** Interactive charts created to represent KPIs and trends.

## How to Navigate
*   Use the **Top Navigation Buttons** to toggle between Overview, Doctor, and Finance views.
*   Analyze the **Area Chart** to understand patient discharge cycles.
*   Check the **Medicine Matrix** to identify peak demand periods for pharmacy stock.
